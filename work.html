3 in a row

3 in a row is a game that I created in my Javascript course while attending NSCC. The goal of the game is to fill the board with red and green such that no colors have a string of 3 in a row but while also having an even number of both colors in each row and column. I'll plan on implementing the code to my website soon so you can play it for yourself!
